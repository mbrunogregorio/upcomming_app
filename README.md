# Upcomming movies App

This Application uses Javascript to create a user interface to show a list of upcomming movies, the data is taken from [TMDb](https://www.themoviedb.org/documentation/api) throught another API Rest.

## Available url

### Upcomming Movies List and Movies search
#### /

### Movie details
#### /details/:id

##### The id params is the movie's Id on TMDb API


## Running application

Necessary Node.js installed
Run the following commands on terminal:
#### yarn
#### yarn start

#### The application will be listening at:
##### http://localhost:3000
